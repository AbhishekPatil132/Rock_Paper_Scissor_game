    <h1>Rock-Paper-Scissors Game</h1>
    <h2>Game Rules</h2>
    <ul>
      <li>Rock beats Scissors</li>
      <li>Scissors beats Paper</li>
      <li>Paper beats Rock</li>
    </ul>
    <h2>Game Interface</h2>
    <p>The game interface consists of the following elements:</p>
    <ul>
      <li>Three buttons for the player to select their move (Rock, Paper, or Scissors)</li>
      <li>Images showing selected options of user and computer</li>
      <li>A message box that displays the outcome of each round (e.g. "You Win!", "You lost!", or "match draw!")</li>
    </ul>
    <h2>Getting Started</h2>
    <p>To play the game, simply open the <code>Game.html</code> file in your web browser.</p>
